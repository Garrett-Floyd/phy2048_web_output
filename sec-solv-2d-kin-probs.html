    Section 7.3 Solving 2D Kinematic Problems

    Lets go over the recommended steps for solving a two dimensional kinematics problem by working an example. A tennis player hits the tennis ball 2.0 meters above the ground. The ball’s initial speed is 20.0 meters per second, directed at 5.0 degrees above the horizontal. The distance to the net is 7.0 meters, and the net is 1.0 meter high. Does the ball clear the net, and if so, how high is the ball when it passes over the net?
    🔗
    Typical solution steps: The setup for the problem is nearly identical to the setup for a multi-object kinematics problem.
    1. Break any given vectors (like velocity vectors) into \(x\) and \(y\) components.
      🔗
      The \(x\)-component of the intial velocity is
      \begin{equation*} v_{x,i} = 20.0 \left( \frac{m}{s} \right) \cos (5^{\circ}) = 19.924 \frac{m}{s} \end{equation*}
      and the \(y\)-component of the intial velocity is
      \begin{equation*} v_{x,i} = 20.0 \left( \frac{m}{s} \right) \cos (5^{\circ}) = 1.743 \frac{m}{s}. \end{equation*}
      🔗
      🔗
    2. List all the \(x\)-givens and unknowns in one table. List all the \(y\)-givens and unknowns in a second table.
      🔗
      We want to analyze what happens at the net so lets make the net our final point. Because \(a_{x} = 0\text{,}\) \(v_{x, f} = v_{x, i}\text{.}\) Since \(v_{x} \gt 0\) the ball will definitely reach the net.
      🔗
      🔗
    3. Circle the unknowns that are equal between the two tables (usually \(t_{f}\) and \(t_{i}\)). For increased accessibility we don’t circle kinematic variables that come out equal between the two tables in these notes, but instead drop the subscripts denoting which object the variable belongs to since it is equal in both.
      🔗
      \(t_{i}\) and \(t_{f}\) are equal between the two tables so we’ve dropped any \(x\) or \(y\) subscripts in Table 7.3.2 and Table 7.3.2.
      🔗
      🔗
    4. Then pick the best equation for the \(x\)-givens, and, separately, pick the best equation for the \(y\)-givens. (You will frequently wind up solving one equation for time, and subbing time back into the other equation.)
      🔗
      For the \(y\)-component we pick the following equation
      \begin{equation*} y_{f} = \frac{1}{2} a_{y} (\Delta t)^{2} + v_{y, i} \Delta t + y_{i}. \end{equation*}
      This is a good choice because it contains \(y_{f}\) which we want to solve for but not \(v_{y,f}\) which we don’t know. It contains \(t_{f}\) which we don’t know but \(t_{f}\) is the only unknown in the \(x\) givens table so we can eliminate it with our \(x\)-component equation. For the \(x\)-component we pick the following equation
      \begin{equation*} x_{f} = \frac{1}{2} a_{x} (\Delta t)^{2} + v_{x, i} \Delta t + x_{i}. \end{equation*}
      This is a good choice because we know everything else except \(t_{f}\)
      🔗
      🔗
    5. Solve the algebra.
      🔗
      Now we plug in all the zero values and solve for \(t_{f}\) in the \(x\)-component equation.
      \begin{align*} x_{f} \amp = v_{x,i} t_{f}\\ \\ t_{f} \amp = \frac{x_{f}}{v_{x,i}} \end{align*}
      This gives a symbolic solution for \(t_{f}\) so we can treat it as a known and use it to solve for \(y_{f}\text{.}\) Considering this and pluggin in all the zero values
      \begin{equation*} y_{f} = \frac{1}{2} a_{y} t_{f}^{2} + v_{y, i} t_{f}^{2} + y_{i}. \end{equation*}
      is already a symbolic solution for \(y_{f}\text{.}\) Now lets plug in numbers
      \begin{equation*} t_{f} = \frac{7.0 m}{19.924 \frac{m}{s}} = 0.35134 s \end{equation*}
      \begin{equation*} y_{f} = \frac{1}{2} (-9.81 \frac{m}{s^2}) (0.35134 s)^{2} + (1.743 \frac{m}{s}) (0.35134 s) + 2 m = 2.01 m \end{equation*}
      \(2.01 m \gt 1.00 m\) so the ball clears the net.
      🔗
      🔗
