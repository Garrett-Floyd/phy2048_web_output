Theorem 16.2.1.

Let \(W_{i, f}^{c}\) be the work done by a conservative force \(\vec{F}_{c}\) as an object is moved from an arbitrary initial point \(i\) to an arbitrary final point \(f\text{.}\) Let \(U\) be the potential energy due to \(\vec{F}_{c}\) with reference point \(P\text{.}\) Then
\begin{equation*} \Delta U = -W_{i, f}^{c}. \end{equation*}
🔗
🔗

Proof.

Consider the work done by \(\vec{F}_{c}\) as an object is moved from \(P\) to \(i\) to \(f\) back to \(P\text{.}\) By Theorem 16.1.2 the work done by \(\vec{F}_{c}\) on this closed loop is zero so we can write
\begin{align*} W_{P, i}^{c} + W_{i, f}^{c} + W_{f, P}^{c} \amp = 0\\ \\ W_{P, i}^{c} + W_{f, P}^{c} \amp = -W_{i, f}^{c}\\ \\ W_{P, i}^{c} - W_{P, f}^{c} \amp = -W_{i, f}^{c}\\ \\ -U_{i} + U_{f} \amp = -W_{i, f}^{c}\\ \\ U_{f} - U_{i} \amp = -W_{i, f}^{c}\\ \\ \Delta U \amp = -W_{i, f}^{c} \end{align*}
