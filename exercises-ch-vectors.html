    Vector Addition and Subtraction

    1. How adding vectors tip-to-tail (graphically) corresponds to adding vectors algebraically.

    Your group should annotate on the graph paper in Figure 2.2.1 to practice adding the specified vectors.
    🔗
    1. Choose a starting point to begin at. Draw the vector \(\vec{A} = \langle +3.0, +1.0 \rangle\) centimeters.
      🔗
      🔗
    2. Continuing onwards from the tip of \(\vec{A}\text{,}\) draw the next leg: the vector \(\vec{B} = \langle +4.0, +5.0 \rangle\) centimeters. We have now arrived at the destination point.
      🔗
      🔗
    3. How many horizontal grid squares is the total run, \(\Delta x\text{,}\) for the entire trip, \(\vec{A} + \vec{B}\text{?}\) How can you calculate that value just from the Cartesian components of \(\vec{A}\) and \(\vec{B}\text{?}\)
      🔗
      🔗
    4. How many vertical grid squares is the total rise, \(\Delta y\text{,}\) for the entire trip, \(\vec{A} + \vec{B}\text{?}\) How can you calculate that value just from the Cartesian components of \(\vec{A}\) and \(\vec{B}\text{?}\)
      \begin{equation*} \vec{A} + \vec{B} = \langle \fillinmath{X} , \fillinmath{X} \rangle \ \text{centimeters} \end{equation*}
      🔗
      🔗
    What you have just done shows that column addition allows you to add vectors algebraically. So, if you have the Cartesian components, you can add them using numbers, without having to physically draw the vectors to scale on graph paper.
    🔗
    In other words, if:
    \begin{equation*} \vec{A} = \langle A_x, A_y \rangle \end{equation*}
    and:
    \begin{equation*} \vec{B} = \langle B_x, B_y \rangle \end{equation*}
    Then the sum is:
    \begin{equation*} \vec{A} + \vec{B} = \langle A_x + B_x, A_y + B_y \rangle \end{equation*}
    🔗
    🔗

    2. Two warnings.

    Let’s check out whether it is true that the magnitude of the resultant vector can be found by adding the magnitudes of the ingredients. In other words, is it true that \(|\vec{A}| + |\vec{B}| = |\vec{A} + \vec{B}|\text{?}\) For this check, use the same vector values as above: \(\vec{A} = \langle 3.0, 1.0 \rangle \ \text{centimeters}\text{,}\) \(\vec{B} = \langle 4.0, 5.0 \rangle \ \text{centimeters}\text{,}\) \(\vec{A} + \vec{B} = \langle 7.0, 6.0 \rangle \ \text{centimeters}\text{.}\)
    🔗
    1. Use Pythagoras to get the magnitude \(| \vec{A} |\text{:}\)
      \begin{equation*} |\vec{A}| = \sqrt{\fillinmath{X}} \end{equation*}
      🔗
      🔗
    2. Use Pythagoras to get the magnitude \(| \vec{B} |\text{:}\)
      \begin{equation*} |\vec{B}| = \sqrt{\fillinmath{X}} \end{equation*}
      🔗
      🔗
    3. Add those magnitudes up to see what \(| \vec{A} |+| \vec{B} |\) is:
      \begin{equation*} | \vec{A} | + | \vec{B} | = \fillinmath{X} \end{equation*}
      🔗
      🔗
    4. Use Pythagoras to get the magnitude of the resultant:
      \begin{equation*} | \vec{A} + \vec{B} | = \sqrt{ \left( 7.0 \text{cm} \right)^2 + \left( 6.0 \text{cm} \right)^2 } = 9.22 \text{cm} \end{equation*}
      🔗
      🔗
    5. Conclude: is it true that \(|\vec{A}|+|\vec{B}|=|\vec{A}+\vec{B}|\text{?}\) Indicate one:
      \begin{equation*} \text{True / False} \end{equation*}
      🔗

    3.

    Let’s check out whether it is true that the angle of the resultant vector can be found by adding the angles of the ingredients. In other words, is it true that \(\theta_{A} + \theta_{B} = \theta_{A + B}\text{?}\) For this check, use the same vector values as above: \(\vec{A} = \langle 3.0, 1.0 \rangle \ \text{centimeters}\text{,}\) \(\vec{B} = \langle 4.0, 5.0 \rangle \ \text{centimeters}\text{,}\) \(\vec{A} + \vec{B} = \langle 7.0, 6.0 \rangle \ \text{centimeters}\text{.}\)
    🔗
    1. Use trigonometry to get the direction of \(\vec{A}\text{:}\)
      \begin{equation*} \theta_A = \arctan \left( \frac{1.0 \text{cm}}{3.0 \text{cm}} \right) = \fillinmath{XXX} \end{equation*}
      🔗
      🔗
    2. Use trigonometry to get the direction of \(\vec{B}\text{:}\)
      \begin{equation*} \theta_A = \arctan \left( \frac{\fillinmath{XXX}}{\fillinmath{XXX}} \right) = \fillinmath{XXX} \end{equation*}
      🔗
      🔗
    3. Add those directions up to see what \(\theta_A + \theta_B\) is:
      \begin{equation*} \theta_A + \theta_B = \fillinmath{XXX} \end{equation*}
      🔗
      🔗
    4. Use trigonometry to get the direction of the resultant vector:
      \begin{equation*} \theta_{A + B} = \arctan \left( \frac{6.0 \text{cm} }{7.0 \text{cm} } \right) = 40.6^{\circ} \end{equation*}
      🔗
      🔗
    5. Conclude: is it true that \(\theta_{A} + \theta_{B} = \theta_{A + B}\text{?}\) Indicate one:
      \begin{equation*} \text{True / False} \end{equation*}
      🔗
      🔗
    You’ve just discovered that there’s no shortcut to find the magnitude or angle of the resultant. You first had to add Cartesian components to get the components of the resultant, \(\langle 7.0, 6.0 \rangle\) centimeters. Only then can you use Pythagoras to find the magnitude and trigonometry to find the direction. In other words, vectors can’t be added in polar form, only in Cartesian form (i.e., if you want a polar form answer, you have to convert from Cartesian to polar after all the addition is done).
    🔗
